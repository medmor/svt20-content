4. Réalisez dans le même repère, les polygones de fréquences des populations (P₂) et (P'₂). (1,25 pts)
Utilisez l'échelle : 1 cm pour 10 plantes et 0,5 cm pour chaque classe.

5. Montrez l'efficacité de la sélection artificielle dans l'amélioration de la productivité de la fève dans chacune des populations (P₁) et (P₂). (1 pt)

Correction

1.

  • Type de variation : Variation quantitative continue (le NGP prend des valeurs numériques dans un intervalle, mesurables). (0,5 pt)

2.

  • Calculs pour P₁ : (1,75 pts)
    n = 5+11+18+26+36+60+51+33 = 240 plantes
    x̄ = Σ(fᵢxᵢ) / n = (21×5 + 22×11 + 23×18 + 24×26 + 25×36 + 26×60 + 27×51 + 28×33) / 240
    x̄ = (105 + 242 + 414 + 624 + 900 + 1560 + 1377 + 924) / 240 = 6146 / 240 = 25,61 gousses/plant
    σ = √[Σfᵢ(xᵢ - x̄)² / n] = 1,87 gousses/plant (calcul détaillé dans tableau)
    Intervalle de confiance : [25,61 - 1,87 ; 25,61 + 1,87] = [23,74 ; 27,48]

3.

  • Population la plus productive : P₁ est plus productive que P₂ car sa moyenne (25,61) est supérieure à celle de P₂ (15,85). (0,5 pt)

4.

  • Polygones de fréquences : Tracer les deux courbes sur le même graphique avec l'échelle donnée. P'₂ montre un décalage vers les NGP élevés par rapport à P₂. (1,25 pts)

5.

  • Efficacité de la sélection artificielle :
    - Pour P₁ : P'₁ a une moyenne plus élevée que P₁ (sélection des plantes avec NGP > 25). (0,5 pt)
    - Pour P₂ : P'₂ a une moyenne plus élevée que P₂ (15,85 → ~25). La sélection des plantes avec NGP > 18 a permis d'obtenir une génération avec une productivité nettement améliorée. (0,5 pt)
